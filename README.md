## 🇬🇧 English

### What is AlsoPlaying?

AlsoPlaying is a Windows tool that generates small `.exe` files for each console or external platform game you're playing. By adding that `.exe` to Steam or Epic Games as an "external game", your friends will be able to see what you're playing even if it's not a PC game.

Perfect for playing on PS5, Nintendo Switch, Xbox, emulators, or any other platform while keeping your Steam or Epic status updated.

### How does it work?

1. Open **AlsoPlaying**
2. Enter the game name and platform
3. The app generates a custom `.exe` on your Desktop
4. Add that `.exe` to Steam or Epic as a non-Steam/external game
5. Every time you run it, a small widget will appear with the game name and a timer

### Requirements

- Windows 10 / 11
- [Python 3.13 or higher](https://www.python.org/downloads/) — with **"Add Python to PATH"** checked during installation
- PyInstaller — AlsoPlaying installs it automatically if not detected

> ⚠️ **Important:** Install Python from [python.org](https://www.python.org/downloads/), **not** from the Microsoft Store.

### Installation & usage

1. Download `AlsoPlaying.exe` from the [Releases](../../releases) section
2. Run `AlsoPlaying.exe` — on first launch it will check for Python and PyInstaller and guide you if anything is missing
3. Fill in the game name and platform
4. (Optional) Select a custom `.ico` icon
5. Click **⚡ GENERAR .EXE** — the file will appear on your Desktop

### Add to Steam

1. Open Steam
2. Go to **Library → Add a Game → Add a Non-Steam Game**
3. Select the generated `.exe`
4. Done! When you run it, Steam will show you as playing that game

### Add to Epic Games

1. Open Epic Games Launcher
2. Go to your **Library**
3. Click the **+** button in the bottom left corner
4. Select the generated `.exe`
5. Done! Your Epic friends will see your game status

## 🇪🇸 Español

### ¿Qué es AlsoPlaying?

AlsoPlaying es una herramienta para Windows que genera pequeños `.exe` para cada juego de consola o plataforma externa que estés jugando. Al añadir ese `.exe` a Steam o Epic Games como "juego externo", tus amigos podrán ver a qué estás jugando aunque no sea un juego de PC.

Perfecto para jugar en PS5, Nintendo Switch, Xbox, emuladores, o cualquier otra plataforma mientras mantienes tu estado de Steam o Epic actualizado.

### ¿Cómo funciona?

1. Abres **AlsoPlaying**
2. Escribes el nombre del juego y la plataforma
3. La app genera un `.exe` personalizado en tu Escritorio
4. Añades ese `.exe` a Steam o Epic como juego externo
5. Cada vez que lo ejecutes, aparecerá un pequeño widget con el nombre del juego y un cronómetro

### Requisitos

- Windows 10 / 11
- [Python 3.13 o superior](https://www.python.org/downloads/) — con la opción **"Add Python to PATH"** marcada durante la instalación
- PyInstaller — AlsoPlaying lo instala automáticamente si no lo detecta

> ⚠️ **Importante:** Instala Python desde [python.org](https://www.python.org/downloads/), **no** desde la Microsoft Store.

### Instalación y uso

1. Descarga el archivo `AlsoPlaying.exe` desde la sección [Releases](../../releases)
2. Ejecuta `AlsoPlaying.exe` — al primer arranque comprobará si tienes Python y PyInstaller instalados y te guiará si falta algo
3. Rellena el nombre del juego y la plataforma
4. (Opcional) Selecciona un icono `.ico` personalizado
5. Pulsa **⚡ GENERAR .EXE** — el archivo aparecerá en tu Escritorio

### Añadir a Steam

1. Abre Steam
2. Ve a **Biblioteca → Añadir un juego → Añadir un juego que no es de Steam**
3. Selecciona el `.exe` generado
4. ¡Listo! Cuando lo ejecutes, Steam mostrará que estás jugando a ese juego

### Añadir a Epic Games

1. Abre Epic Games Launcher
2. Ve a tu **Biblioteca**
3. Pulsa el botón **+** en la esquina inferior izquierda
4. Selecciona el `.exe` generado
5. ¡Listo! Tus amigos de Epic verán tu estado de juego
